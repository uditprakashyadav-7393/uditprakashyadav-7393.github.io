## 💻 Usage

### Quick Start

1. **Copy the HTML** - Include the button markup in your HTML file:

```html
<button class="btn btn-primary">Hover Me</button>
```

2. **Link the CSS** - Add the stylesheet to your HTML:

```html
<link rel="stylesheet" href="styles.css">
```

3. **Customize** - Modify colors, sizes, and animations in `styles.css`

### Available Button Classes

- `.btn-primary` - Primary action button
- `.btn-secondary` - Secondary action button
- `.btn-success` - Success/confirmation button
- `.btn-danger` - Danger/delete button
- `.btn-info` - Information button
- And more variants...

---

## 🎨 Customization

All button styles are easily customizable through CSS variables and classes:

```css
/* Modify colors */
.btn-primary {
  background-color: #your-color;
  color: #text-color;
}

/* Adjust animation speed */
.btn {
  transition: all 0.3s ease;
}
```

3. **Mobile** - Test animations on different devices
4. **Consistency** - Use similar timing and easing functions across buttons
